Portfolio Website – Overview
This is a responsive personal portfolio website developed using HTML5 and CSS3. It is designed to present personal, academic, and professional information in a clean, modern, and user-friendly interface.

Key Features

Navigation Bar: A sticky navigation menu with smooth scrolling for easy access to different sections.
Hero Section: Displays the name, educational details, and a call-to-action button.
About Section: Provides a brief introduction, educational background, and career interests.
Skills Section: Highlights technical skills using interactive skill badges with hover effects.
Projects Section: Showcases completed projects inside a modern card layout.
Contact Section: Includes a contact form for visitors to send messages.
Footer: Displays copyright information.

Design Highlights
Uses a dark cyber-themed gradient background with glowing animated effects for a premium appearance.
Implements glassmorphism-inspired cards, smooth hover animations, and modern typography using the Poppins font.
Fully responsive layout that adapts to different screen sizes.

Technologies Used
HTML5 – Structure of the webpage.
CSS3 – Styling, animations, responsiveness, and visual effects.
Google Fonts – Poppins font for modern typography.
